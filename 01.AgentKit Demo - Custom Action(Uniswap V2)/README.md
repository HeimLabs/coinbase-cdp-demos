## Setup

1. Clone this repository
2. Install dependencies:
   ```
   npm install
   ```
3. Copy `.env.example` to `.env` and add your OpenAI API key:
   ```
   cp .env.example .env
   ```
4. Edit the `.env` file and add the needed keys

5. Run the chatbot:
   ```
   npm start
   ```

6. Run the chatbot in development mode:
   ```
   npm run dev
   ```
